# Beyond Kubernetes

There are many tools that use Kubernetes. OpenShift is an "enterprise" Kubernetes. There's also Rancher, Anthos GKE, etc. All viable options when we want to use a managed Kubernetes.

### Serverless

Serverless is rising in popularity. We will set up a serverless platform on top of our k3d. It will use Knative as it is what Google Cloud Run is based on. For an app to run correctly on Knative, it must be stateless and configurable with environment variables.
